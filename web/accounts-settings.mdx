---
title: Accounts Settings
description: Configure new user Signup Bonus and OAOD (one account for one device)
---

Follow these steps to configure new user Signup Bonus and OAOD (one account for one device) settings :

  1. Log in to your Admin Interface (Pocket Script Instance)
  2. Navigate to `Settings --> Configuration --> Global Settings` from Admin Sidebar
  3. Scroll down to the `Registration/Signup Bonus` Section to configure Signup Reward
  4. Scroll down to the `OAOD` Section to configure one account for one device

That's It!

---
title: Proxy Detection
description: Configure Proxy/VPN Detection settings
---

Follow these steps to configure Proxy/VPN Detection settings :

  1. Log in to your Admin Interface (Pocket Script Instance)
  2. Navigate to `Settings --> Configuration --> VPN/Proxy Detection` from Admin Sidebar
  3. Choose VPN/Proxy Detection Service
  4. Finally, provide API key for the choosen service

That's It!

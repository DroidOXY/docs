## Overview

After purchasing Pocket Script, you'll receive a ZIP archive containing all the necessary files to set up your rewards platform. This guide will walk you through the contents of your download and explain the purpose of each component.

## Unpacking Your Pocket Script Files

Follow these simple steps to access your Pocket Script files:

1. **Download the ZIP archive**: After your purchase, you'll receive a link to download the Pocket Script package.
2. **Locate the download**: Find the ZIP file in your computer's download folder or the location you specified.
3. **Extract the contents**: Use your computer's built-in unzip tool or a third-party application like 7-Zip to extract the files.

<Tip>
  If you're unsure how to extract ZIP files on your operating system, a quick web search for "how to unzip files on [Your OS]" will provide step-by-step instructions.
</Tip>

## Understanding the File Structure

Once you've extracted the ZIP archive, you'll find the following components:

### 1. documentation.pdf

This PDF file contains a direct link to the comprehensive online documentation you're currently reading. It's a quick way to access the most up-to-date information about Pocket Script.

<Info>
  Pro tip: Bookmark the documentation link for easy access in the future!
</Info>

### 2. support.pdf

If you encounter any issues during installation or have questions about using Pocket Script, this PDF is your go-to resource. It provides information on how to get support from our team.

<Note>
  We're here to help! Don't hesitate to reach out if you need assistance at any point.
</Note>

### 3. New upload folder

This is the main source code of the Pocket Script package. This folder contains all the core files required to set up and run Pocket Script on your server. Here's what you need to know about it:

- It includes the main application files, admin dashboard, user dashboard and main website theme.
- You'll need to Upload the Contents of New upload folder to your server
- Detailed instructions for handling these files can be found in our [Installation Guide](/get-started/installation).

### 3. update folder

This folder contains updated files that needs to be uploaded to your server depending on the version of the Pocket Script you are using. Here's what you need to know about it:

- This folder files are not Needed for FRESH Installation
- This folder files are used to update the Pocket Script to the latest version
- Detailed instructions for handling these files can be found in our [How to Update Guide](/get-started/how-to-update-pocket-script).

<Warning>
  Always Make A Backup of your Server Files and Database before updating the script. You may need it incase something goes wrong.
</Warning>
